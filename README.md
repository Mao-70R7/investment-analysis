# investment-analysis

Static investment advisory analysis outputs and data pages.

## Contents

- `basic_data/`: static site pages, assets, and generated data files.
- `analysis_outputs/`: generated reports, source scripts, CSV, and JSON outputs.
- `output/pdf/`: PDF deliverables and rendered page previews.
- `全市场投顾监控数据与页面系统说明_Codex接入指南.md`: database schema, model logic, page data structure, and Codex analysis guide.

Open `index.html` or `basic_data/index.html` in a browser to view the generated pages.
